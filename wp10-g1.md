## 0. Setup and notation

L = log x; P = x^u, Q = x^{u'}, u,u' in (1/3,1/2); R = x^{1-u-u'};
band facts used (all numerically re-verified in §5.4): u+u' in (2/3,1),
2u+u' >= 1, u+2u' >= 1 on the closed band, hence R <= min(P,Q);
M_0 := R L^{-B}; delta_nu := 1/(2 nu L^B); lambda integer in [1, L^C].

prop:two-freq gives phases theta_mu = mu q/p - lambda p/q (mod 1) for
0 < mu < p, weight |f_mu| <= 1/(2<mu>), <mu> = min(mu, p-mu). (M1) at
level B demands, for EVERY mu with 0 < <mu> <= R:
min(R, 1/(2||theta_mu||)) <= <mu> L^B.

Failure bookkeeping, exact:
- (M1) fails at mu iff ||theta_mu|| < 1/(2 <mu> L^B) AND <mu> < R L^{-B}
  (the min(R,·) cap makes <mu> >= R L^{-B} immune). So only
  nu := <mu> <= M_0 matters.
- LOWER half (mu = nu <= M_0): theta_mu = nu q/p - lambda p/q — the
  DIFFERENCE branch g_-(p) = nu q/p - lambda p/q. This is lem:branch /
  wp8-e1 territory: g_- is strictly decreasing in p (g_-' =
  -nu q/p^2 - lambda/q < 0), no critical point, one interval per
  (q,nu,k). PROVED machinery; untouched here.
- UPPER half (mu = p - nu, nu <= M_0): since p(q/p) = q in Z,
  theta_{p-nu} = (p-nu)q/p - lambda p/q ≡ -(nu q/p + lambda p/q) (mod 1),
  so ||theta_{p-nu}|| = ||g_+(p)||, g_+(p) := nu q/p + lambda p/q —
  the SUM branch. NOTE: p prime > nu guarantees p ∤ nu q only via
  p ∤ q... not needed: the identity is unconditional (q/p·p = q ∈ Z).

Sum-branch structure (the whole point of G1):
g_+'(p) = lambda/q - nu q/p^2 vanishes at p* := q sqrt(nu/lambda);
g_+'' = 2 nu q/p^3 > 0, so g_+ is strictly CONVEX with q-INDEPENDENT
minimum value
    t* := g_+(p*) = 2 sqrt(lambda nu)  (>= 2 since lambda, nu >= 1).
Exact second-order form (no remainder): with p = p*(1+v),
    g_+(p) - t* = sqrt(lambda nu) · v^2/(1+v).            (T-exp)
[Check: (1+v) + 1/(1+v) = 2 + v^2/(1+v).]

## 1. Tangency-frequency count

Fix q in (Q,2Q] and lambda <= L^C. The tangency point p* = q sqrt(nu/lambda)
falls in the dilated band [P/2, 4P] iff
    nu in M_T(q,lambda) := [lambda P^2/(4q^2), 16 lambda P^2/q^2]
      ⊆ [m_*/16, 16 m_*],   m_* := lambda P^2/Q^2.
Outside M_T the band (P,2P] lies entirely on one monotone side of p*
with non-degenerate slope (Lemma S1 below). So "tangency frequencies"
live in a single fixed dyadic neighbourhood nu ≍ m_* of length
<= 16 m_*, and they exist at all only if

  (T-exist)  m_*/16 <= M_0  and  Q <= 4P sqrt(lambda)·(const),

i.e. lambda x^{2u-2u'} <= 16 R L^{-B} and x^{u'-u} <= 4 sqrt(lambda)
(the second because p* >= q/sqrt(lambda) > 4P kills all nu >= 1).
Consequences: on the half-band u' > u the strips exist only inside the
sliver u' - u <= (C/2 + O(1)) loglog x/log x — wp9-verify-e1e2's
"u >= u' half-band only" claim, made precise; and at the
(1/2-,1/3)-type corner m_* > 16 M_0 (since 3u - u' - 1 > 0 there) so
the tangency range is EMPTY (numerics §5.3).

**Lemma T (spacing count).** For Delta in (0, 1/2] and m >= 1,
  nu(Delta; m) := #{nu in [m/16, 16m] ∩ Z : ||2 sqrt(lambda nu)|| <= Delta}
  <= c0 ( Delta·m + min(m, sqrt(lambda m)) + 1 ),
with c0 absolute.

*Proof.* The map nu -> t(nu) = 2 sqrt(lambda nu) is increasing with
derivative t' = sqrt(lambda/nu) in [ (1/4)sqrt(lambda/m), 4 sqrt(lambda/m) ]
on the range. Its image is an interval of length <= 2 sqrt(lambda)
(4sqrt(m) - sqrt(m)/4) <= 8 sqrt(lambda m), so the number of integers k
with |t(nu) - k| <= 1/2 for some admissible nu is <= 8 sqrt(lambda m) + 1.
For each such k, {nu : |t(nu) - k| <= Delta} is an interval of t-length
2 Delta, hence of nu-length <= 2 Delta / min t' <= 8 Delta sqrt(m/lambda),
containing <= 8 Delta sqrt(m/lambda) + 1 integers. Multiplying and adding:
nu(Delta) <= (8 sqrt(lambda m) + 1)(8 Delta sqrt(m/lambda) + 1)
<= 64 Delta m + 8 sqrt(lambda m) + 8 Delta sqrt(m/lambda) + 1; the third
term is <= 8 Delta m (lambda >= 1... sqrt(m/lambda) <= m since m >= 1).
Also trivially nu(Delta) <= 16m. Combining gives the min form. ∎

Sharpness note (adversarial): the "+min(m, sqrt(lambda m))" term is NOT
removable — for lambda = 1, every perfect square nu = j^2 in [m/16,16m]
is an EXACT tangency (t = 2j), and there are ≍ sqrt(m) of them. So
"O(1) exact tangencies per lambda" (the phrase in wp9-verify-e1e2's
repair sketch) is FALSE as stated; there are up to ≍ min(m_*,
sqrt(lambda m_*)) of them. The repair survives anyway: each individual
strip has width ∝ nu^{-1/4} delta_nu^{1/2}-scale (§2), so MANY narrow
strips total the same as the verifier's few wide ones — the assembled
budget (§4.1) comes out at the predicted P L^{-B/2}, and the min(m, ·)
refinement of Lemma T is exactly what avoids an L^{C/4} loss on the
u' > u sliver.

## 2. Major-p strip count: the sibling of lem:branch

**Lemma S1 (slopes off the tangency zone).** Let Z := [p*/2, 2p*].
For p in (P,2P] \ Z: if p > 2p*, g_+'(p) = (lambda/q)(1 - (p*/p)^2) in
[(3/4) lambda/q, lambda/q]; if p < p*/2, -g_+'(p) = (nu q/p^2)(1 -
(p/p*)^2) in [(3/4) nu q/p^2, nu q/p^2]. In particular if p* < P/2 the
whole band is increasing with slope ≍ lambda/q (and then nu q/P^2 <=
lambda/(4q), so the lem:branch-form slope scale (nu Q/P^2 + lambda/Q)
is ≍ lambda/Q); if p* > 4P the whole band is decreasing with slope
≍ nu q/p^2 (and lambda/Q <= nu q^2/(16 P^2 Q) ≍ nu Q/P^2). ∎ (direct
computation; the parentheticals from p*^2 = nu q^2/lambda.)

**Lemma S2 (window widths inside Z).** Fix k in Z with
D := k - t* >= 0 (windows with k < t* - delta are empty since
g_+ >= t*). By (T-exp), {p in Z: |g_+(p) - k| <= delta} is contained in
{p = p*(1+v): v in V_-(k) ∪ V_+(k)}, two intervals (one per sign of v),
each of p-length
  <= 8 p* delta (lambda nu)^{-1/4} max(D, delta)^{-1/2},
and the outermost solutions satisfy |v| <= 4 sqrt(max(D,delta)+delta)
(lambda nu)^{-1/4}.

*Proof.* F(v) := sqrt(lambda nu) v^2/(1+v) on v in [-1/2, 1] is 0 at 0,
increasing in |v| on each side, with F(v) >= sqrt(lambda nu) v^2/2 for
v in [-1/2,1]; so F(v) <= D + delta forces v^2 <= 2(D+delta)/sqrt(lambda nu).
If D <= delta (the tangency window itself): the window is {F <= D+delta}
⊆ {|v| <= 2 sqrt(delta) (lambda nu)^{-1/4}}, total p-length <=
4 p* sqrt(delta)(lambda nu)^{-1/4} <= the claim with max(D,delta)=delta.
If D > delta: the window is an F-annulus {D - delta <= F <= D + delta};
on each side |F'(v)| = sqrt(lambda nu)|v|(2+v)/(1+v)^2 >=
(1/2) sqrt(lambda nu) |v| for v in [-1/2, 1]. For D >= 2 delta: every
point of the window has F >= D - delta >= D/2, and F <= sqrt(lambda nu)
v^2·2 on the negative side / <= sqrt(lambda nu) v^2 on the positive
side (1+v >= 1/2 resp. >= 1), so |v| >= (1/2) sqrt(D/sqrt(lambda nu))
throughout the window; hence |F'| >= (1/4)(lambda nu)^{1/4} sqrt(D) on
the window and its v-length is <= 2 delta/min|F'| <=
8 delta (lambda nu)^{-1/4} D^{-1/2}. For delta < D < 2 delta use the
D <= delta computation with delta -> 2 delta (constants absorbed). ∎

**Lemma B+ (sum-branch count; the sibling of lem:branch).**
Fix q in (Q,2Q], integers lambda, nu >= 1, and delta in (0, 1/2]. Let
N_q^+(nu,lambda,delta) := #{p in (P,2P] ∩ Z: ||nu q/p + lambda p/q|| <= delta}.
Then
  N_q^+ = 2 delta P + E_q^+,
  |E_q^+| ≪ nu Q/P + lambda P/Q + 1
           + delta (nu Q/P^2 + lambda/Q)^{-1}
           + 1_{p* in [P/2, 4P]} · q · delta · nu^{1/4} lambda^{-3/4}
             · max(||2 sqrt(lambda nu)||, delta)^{-1/2},
with absolute implied constants. The first four terms are IDENTICAL to
lem:branch; the fifth is the tangency term T(q; nu, lambda, delta).

*Proof.* Case p* not in [P/2,4P]: (P,2P] sits on one monotone side of
p* with the two-sided slope bounds of Lemma S1; the proof of lem:branch
(branch decomposition: #k <= total variation + 2 ≪ nu Q/P + lambda P/Q
+ 1; one interval per k, length = 2delta/|g_+'(xi)| with the slope ≍
its lem:branch-form scale by S1's parentheticals; O(1) endpoint
integers per window; <= 2 clipped windows at the range ends giving the
delta·(slope scale)^{-1} term) transfers verbatim. The only change is
the root formula: g_+(p) = t solves p = c^{(±)}(t) q,
c^{(±)}(t) = (t ± sqrt(t^2 - 4 lambda nu))/(2 lambda) — the SMALLER
root c^{(-)} on the decreasing side (p < p*, i.e. c < sqrt(nu/lambda)),
the larger root c^{(+)} on the increasing side (p > p*) — endpoints
again LINEAR in q with q-independent slopes (consumed in §3).
Case p* in [P/2,4P]: split (P,2P] = (band ∩ Z) ∪ (band \ Z). On
band \ Z (at most two monotone pieces, on each S1 applies) the same
lem:branch accounting holds; the clipped-window terms at the Z-boundary
have slope >= (1/4)(lambda/q)(lambda nu)^{... } — at |v| = 1/2 the slope
is >= (1/8) sqrt(lambda nu)/p* · p*-scale... bounded below by the D ≍
sqrt(lambda nu)/8-height of S2, so those two windows have length
≪ p* delta (lambda nu)^{-1/4} (sqrt(lambda nu))^{-1/2} ≪ delta p*
(lambda nu)^{-1/2} <= 2 delta P — absorbed into the main-term constant.
On band ∩ Z: the integers k with a nonempty window satisfy
t* - delta <= k <= t* + F(1) + delta <= t* + sqrt(lambda nu), so they
are k_j with heights D_j := k_j - t* >= -delta, D_0 := max(||t*|| - 0,
0) handled as: the at most ONE k in [t* - delta, t* + delta] exists iff
||t*|| <= delta and contributes (S2, max = delta) length
<= 8 p* sqrt(delta) (lambda nu)^{-1/4} = (T-term at max(||t*||,delta) =
delta, since p* = q sqrt(nu/lambda) gives p* (lambda nu)^{-1/4} =
q nu^{1/4} lambda^{-3/4}); the k's above t* + delta have heights
D_0' >= max(||t*||, delta) (the nearest integer above t*, if not within
delta, is at height >= ||t*||) followed by D_0' + j, j = 1, 2, ...,
J <= sqrt(lambda nu); by S2 their total length is
  <= 8 p* delta (lambda nu)^{-1/4} [ max(||t*||,delta)^{-1/2}
     + sum_{j>=1} j^{-1/2} ]
  <= 8 p* delta (lambda nu)^{-1/4} max(||t*||,delta)^{-1/2}
     + 8 p* delta (lambda nu)^{-1/4} · 2 sqrt(J)
     [sum_{j=1}^{J}(D_0'+j)^{-1/2} <= sum_{j=1}^{J} j^{-1/2}
      <= 2 sqrt(J), J <= sqrt(lambda nu)]
  <= T-term + 16 p* delta <= T-term + 64 delta P.
Integer count = total length + O(1) per nonempty window; the number of
nonempty windows in Z is <= sqrt(lambda nu) + 2 ≪ nu Q/P + lambda P/Q + 1
(AM-GM: sqrt(lambda nu) = sqrt((nu Q/P)(lambda P/Q)) <= half their sum).
Collecting terms gives the lemma. ∎

Remarks (adversarial notes to the future verifier):
(i) The count is over INTEGER p — an upper bound for prime p; familyA
only excludes the major set, so this is the needed direction (same as
lem:branch).
(ii) Strips clipped by the band edges only help (upper bound).
(iii) delta = delta_nu = 1/(2 nu L^B) <= 1/2 always (nu, L^B >= 1).
(iv) At exact tangency (||t*|| = 0, lambda nu a perfect square of a
half-integer... i.e. 4 lambda nu = k^2) the T-term is
q sqrt(delta) nu^{1/4} lambda^{-3/4}: for the model in §5.1 (lambda =

(v) The lemma is FALSE without the T-term: at exact tangency the
